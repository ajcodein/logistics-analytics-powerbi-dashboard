# Logistics Analytics Dashboard | Power BI

## Project Overview

This project is an end-to-end Logistics Analytics Dashboard developed using Power BI to analyze shipment performance, transportation costs, inventory management, warehouse utilization, customer trends, and operational efficiency.

## Dashboard Pages

### Overview

* Total Revenue
* Total Shipments
* Total Freight Cost
* On-Time Delivery %

### Shipment Analysis

* Top 10 Routes by Revenue
* Average Delivery Delay by Route
* Freight Cost by Vehicle Type
* Customer-wise Shipment Count

### Inventory & Warehouse

* Opening vs Closing Stock Trend
* Inventory Value by Warehouse
* Stock In vs Stock Out
* Warehouse Utilization KPI

### Customer & Product

* Revenue by Customer Type
* Revenue by Product Category
* Average Shipment Weight
* Dynamic Metric Switching

### Operations

* Driver Rating vs Delivery Delay
* Freight Cost by Fuel Type
* Freight Cost by Vehicle Type
* Custom Tooltip Page
* Bookmarks & Navigation

## Technologies Used

* Power BI

* Data Visualization
